# Discord Payment Selector Bot
A simple discord bot made for selecting preferred payment method

# Installation
1. Install [Node.js 18](https://nodejs.org/)
2. Run ```npm i ```
3. Fill in the config file
4. Run ```node .```
Enjoy!
